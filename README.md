# minesweeper-vim
Curses-based minesweeper with (sort of) vim bindings

Controls
       [Up]             [k]
[Left][Down][Right]  [h][j][l]

[x] = sweep cell
[m] = mark flag

[Backspace] = left
    [Space] = right
   [Return] = beginning of next row
        [0] = beginning of row
        [$] = end of row
        [w] = next unswept row
        [b] = previous unswept row
        [H] = beginning of 1st row
        [M] = beginning of middle row
        [L] = beginning of bottom row
